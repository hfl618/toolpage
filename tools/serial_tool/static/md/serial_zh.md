# 云端串口调试 Pro

## 功能特性
- **Web Serial API**: 无需驱动（除硬件本身驱动），在 Chrome/Edge 中直接访问物理串口。
- **极速性能**: 支持最高 2,000,000 波特率。
- **HEX 模式**: 支持 HEX 接收与 HEX 发送（自动解析空格）。
- **安全性**: 所有数据在本地浏览器处理，不经过云端服务器。

## 使用说明
1. 插入串口设备（如 CP2102, CH340）。
2. 在左侧控制中心选择正确的波特率。
3. 点击 **开启串口连接**，在弹窗中选择对应的设备。
4. 通过底部输入框发送指令。
