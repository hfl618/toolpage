# Serial Studio Pro

## Features
- **Web Serial API**: Access hardware serial ports directly in Chrome/Edge.
- **High Performance**: Supports up to 2,000,000 Baud.
- **HEX Mode**: Full support for HEX RX/TX.
- **Privacy**: All data is processed locally.

## Usage
1. Connect your serial device.
2. Select Baud rate in the Control Center.
3. Click **Start Connection**.
