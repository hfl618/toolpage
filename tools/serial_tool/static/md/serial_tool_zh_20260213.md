# 云端串口调试 Pro - 深度工程指南 (v2.5)

## 1. 核心功能概览
- **Web Serial API**: 直接在浏览器中与硬件交互，支持最高 921,600 波特率。
- **模拟仿真模式**: 开启后可生成虚拟传感器数据，测试波形、回复及过滤逻辑。
- **隐私保护**: 硬件数据仅在浏览器本地内存处理，本站不存储任何通讯日志。

## 2. 专业级调试技巧

### 2.1 高级时间戳
- **绝对时间**: 显示标准的本地系统时间。
- **相对时间 (Δ)**: 显示自上一条消息以来的时间差（毫秒）。适用于测量传感器采样频率或指令响应延迟。
- *操作*: 在终端顶部切换 `ABS TIME` / `DELTA`。

### 2.2 多通道波形图 (Waveform)
系统支持自动从原始数据流中提取数字并绘图：
- **普通模式**: 串口输出数字即可（如 `25.5`）。
- **多通道模式**: 串口输出 `Key:Value` 格式（如 `temp:25.5,humi:60`）。
- **颜色对应**: 系统会自动为不同的 Key 分配颜色并生成图例。

### 2.3 关键词着色器 (Highlighter)
实时监测日志，并对匹配的行进行着色：
- **预设**: 自动高亮 `ERROR` (红), `SUCCESS` (绿), `WARN` (黄)。
- **自定义**: 点击侧边栏着色器图标，添加您的业务关键词。

## 3. 自动化与效率

### 3.1 自定义宏指令
| 功能 | 说明 | 示例 |
| :--- | :--- | :--- |
| **文本模式** | 支持转义符 `\r\n` | `AT+GMR\r\n` |
| **HEX 模式** | 发送字节流 | `AA BB 01` |
| **周期发送** | 设定毫秒间隔，实现自动查询 | `1000ms` |

### 3.2 自动回复规则 (Auto-Responder)
- **匹配**: 一旦接收到的数据包含关键词，立即触发。
- **响应**: 支持自动回发文本或 HEX 指令（毫秒级响应）。

## 4. 数据管理
- **备份与还原**: 宏指令和规则支持导出为 JSON 文件，方便在不同电脑间迁移配置。
- **运行日志导出**: 一键将当前终端内容保存为 `.txt` 文件。
