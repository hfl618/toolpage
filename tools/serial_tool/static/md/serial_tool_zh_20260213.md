# 云端串口调试 Pro - 深度工程指南 (v2.5)

## 1. 核心技术优势 (SEO)
- **Web Serial API 串口调试工具**: 基于现代浏览器原生 API，实现直接与硬件交互，支持最高 **2,000,000** 极速波特率。
- **免安装浏览器串口助手**: 无需安装任何驱动或 .exe 插件，只要使用 Chrome 或 Edge 浏览器，即可在 Windows、macOS 及 Linux 上快速建立连接。
- **隐私与安全**: 遵循浏览器沙盒机制，数据处理 100% 发生在用户本地，本站服务器不接触、不存储任何串口通讯内容。

## 2. 深度调试技巧

### 2.1 高级时间戳模式
- **绝对时间 (ABS)**: 显示标准的本地 24 小时制时间（如 `15:30:05`）。
- **相对增量 (DELTA Δ)**: 显示自上一条消息以来的时间间隔（毫秒）。**点击日志前的时间戳**可瞬间在两种模式间自由切换。
- **连续计算**: 即使在绝对时间模式下，后台也会精准更新计时器，确保切换到增量模式时数值实时准确。

### 2.2 多通道实时波形图
系统支持高性能 Canvas 渲染，可实时从原始数据中提取数值：
- **匹配格式**: 支持 `Value` 纯数字或 `Key:Value`（如 `temp:25.5,humi:60`）键值对。
- **自动图例**: 动态识别多个传感器通道，并自动分配专业配色。

### 2.3 智能关键词着色器 (Highlighter)
实时监测日志流，通过色彩快速定位异常或成功节点：
- **自动分配**: 添加关键词时系统会自动轮换 5 种专业配色（红、绿、蓝、橙、紫）。
- **自选颜色**: 在侧边栏**点击已添加的关键词**，可循环切换其背景颜色。
- **极速匹配**: 内部采用转换缓存优化，即使在极高数据频率下，高亮逻辑也不会造成 CPU 延迟。

## 3. 自动化开发流

### 3.1 增强型指令宏
- **换行符支持**: 文本模式支持 `CRLF (\r\n)`、`LF (\n)` 以及 `NONE` 模式，适配所有 AT 指令集。
- **HEX 模式**: 支持标准的十六进制字节流输入（自动过滤空格）。
- **周期轮询**: 设定毫秒级间隔，可模拟心跳包或状态查询。

### 3.2 自动回复规则 (Responder)
- **毫秒级响应**: 逻辑在浏览器主线程运行，匹配关键词后立即回发指定指令。
- **使用方法**:
    1. 点击侧边栏 **Responder** 标题右侧的 `+` 号图标。
    2. **匹配内容**: 输入期望收到的字符串（如 `PING`）。
    3. **回复内容**: 输入想要自动回发的指令（如 `PONG\r\n`）。
    4. **高级支持**: 回复内容支持 `\r`, `\n` 等转义符，适用于所有标准 AT 指令握手。
- **协议仿真**: 配合“仿真模式（SIM）”，可在无硬件环境下模拟完整的双向通讯协议。

## 4. 配置与数据
- **云端同步**: 域名托管在 Cloudflare，结合香港 CN2 高速线路，确保工具全球稳定访问。
- **本地持久化**: 高亮规则、宏指令、自动回复均加密存储在浏览器 `localStorage` 中，刷新页面不丢失。
- **日志导出**: 一键导出带有时间戳的通讯原始记录。
