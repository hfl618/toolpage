# LVGL 图像转换工具使用指南

欢迎使用专业级 LVGL 嵌入式图像转换工具。本工具旨在帮助开发者快速将通用图像格式转换为 LVGL 库可识别的 C 数组或二进制文件。

## 🚀 支持的格式
- **输入格式**：PNG (推荐), JPG, BMP。
- **输出格式**：C 数组 (.c), 二进制文件 (.bin)。

## 🎨 颜色格式说明
- **AUTO**: 智能分析原图色彩。若颜色少于 256 色，将自动转为 **Indexed (I1/I2/I4/I8)** 格式以极大地节省空间。
- **RGB565**: 16位色，适用于大多数单片机屏幕（TFT/LCD）。
- **RGB565A8**: **强力推荐！** 2字节颜色 + 1字节透明度。在保持 16 位色的同时，拥有完美的半透明抗锯齿边缘。
- **ARGB8888**: 32位全彩，包含 8 位 Alpha 通道，显示效果最好但最占内存。

## ⚙️ 核心功能
### 1. 尺寸自定义 (Resampling)
您可以指定目标宽度或高度。系统采用 **Lanczos 高质量重采样算法**。在服务器端缩放比在 MCU 端调用 `lv_img_set_zoom` 更加清晰且节省性能。

### 2. 图像抖动 (Dithering)
当从 24 位真彩色转为 16 位 RGB565 时，由于位深降低，色彩过渡处可能出现“色带”。开启抖动后，系统会通过算法模拟中间色，使视觉效果更加平滑。

### 3. Alpha 预乘 (Premultiply)
开启后，颜色值会预先乘以透明度。这可以显著提升 LVGL 在渲染带透明度的大图时的速度，减轻 CPU 负担。
