LVGL 图像转换工具使用指南
==========================

欢迎使用专业级 LVGL 嵌入式图像转换工具。本工具旨在帮助开发者快速将通用图像格式转换为 LVGL 库可识别的 C 数组或二进制文件。

1. 支持的格式
--------------
- 输入格式：PNG (推荐), JPG, BMP。
- 输出格式：C 数组 (.c), 二进制文件 (.bin)。

2. 颜色格式说明
----------------
- AUTO: 智能分析原图色彩。若颜色少于 256 色，将自动转为 Indexed (I1/I2/I4/I8) 格式以极大地节省空间。
- RGB565: 16位色，适用于大多数单片机屏幕（TFT/LCD）。
- RGB565A8: [强力推荐] 2字节颜色 + 1字节透明度。在保持 16 位色的同时，拥有完美的半透明抗锯齿边缘。
- ARGB8888: 32位全彩，包含 8 位 Alpha 通道，显示效果最好但最占内存。

3. 核心功能
------------
- 尺寸自定义: 您可以指定目标宽度或高度。系统采用 Lanczos 高质量重采样算法。在服务器端缩放比在 MCU 端缩放更加清晰且节省性能。
- 图像抖动: 当从 24 位真彩色转为 16 位 RGB565 时，开启抖动后，系统会使视觉效果更加平滑。
- Alpha 预乘: 开启后，颜色值会预先乘以透明度，可显著提升 LVGL 渲染速度。

4. 隐私与限制
--------------
- 不留痕迹：您的图片在转换完成后会立即从服务器销毁，不上传云端或存入数据库。
- 游客额度：游客每日可免费转换 5 次。登录后可享受更高额度。

提示：如果您的图片是 SVG 格式，请先将其导出为 PNG 再进行转换。
