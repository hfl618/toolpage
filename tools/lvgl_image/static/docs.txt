LVGL 图像转换工具 - 指南 (v1.0 - 2026-2-12更新)
========================================

本工具专门针对嵌入式设备性能受限、显存敏感等特性设计，支持高度自定义的资产导出。

1. 🚀 核心功能与高级特性
-------------------------
- 跨行对齐 (Stride Align): 
  支持 1~64 字节对齐。对于使用 DMA 传输的硬件（如 SPI/LTDC 接口），通常建议设置为 4 或 16 字节对齐，以防止图像水平位移或花屏。
  
- 智能背景填充:
  当将带透明度的 PNG 转换为不透明格式（如 RGB565）时，可在界面指定颜色进行底色合成，解决透明处变黑的问题。

- 重采样 (Resampling):
  采用 Lanczos 高质量缩放算法。在服务端完成缩放比在 MCU 端动态缩放效果更佳，且能节省大量 RAM 占用。

- 特殊 RAW 模式:
  直接将 PNG/JPG 二进制数据打包。这是最省 Flash 的方式，但要求单片机端开启 LVGL 运行时解码器。

2. 🎨 颜色格式 (Color Format) 说明
----------------------------------
- RGB565: 16位色 (R5, G6, B5)，最通用的嵌入式格式。
- RGB565A8: [推荐] 16位颜色 + 独立 8位透明度层。兼顾 16位屏性能与完美半透明效果。
- ARGB8888: 32位全彩，包含完整透明通道，画质巅峰。
- AL88: 8位灰度 + 8位透明度，非常适合单色屏或图标素材。
- Indexed (I1/I2/I4/I8): 索引颜色。通过自适应调色板大幅压缩大小，适合 UI 图标。

3. 🛠️ 兼容性与导出
-------------------
- 版本支持: 支持 LVGL v8 与 v9。会自动根据选择调整 C 语言结构体定义（lv_img_dsc_t / lv_image_dsc_t）。
- 自定义命名: 支持自定义 C 变量名，方便在 UI 代码中直接引用。

4. 🛡️ 隐私与配额
-----------------
- 安全销毁: 图片数据在处理后立即删除，不持久化、不存云端。
- 动态额度:
  - 游客: 享有免费试用额度 
  - 登录用户: 享有完整免费额度。

提示：转换大图且开启 RLE/LZ4 压缩时，转换速度可能稍慢，请耐心等待。
